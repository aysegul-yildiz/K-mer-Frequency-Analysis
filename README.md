# K-mer-Frequency-Analysis
This project was done for cs 202 course and it implements a pointer based binary search tree (BST) in C++ to analyze the frequency of k-mers within a given text file. A k-mer is a sequence of consecutive, non-overlapping characters. 
BSTNode:Defines the node structure (key, frequency, child pointers)
BST:	the core BST logic (insert, delete, traverse).
KmerTree:	high level wrapper for k-mer generation and analysis.
main.cpp:	entry point for testing and time performance analysis.
Makefile:Handles compilation on linux/unix systems
